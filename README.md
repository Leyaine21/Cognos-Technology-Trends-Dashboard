# Cognos-Technology-Trends-Dashboard

# project overview
This project analyses technology survey data using IBM Cognos analytics to identify current technology usage patterns, future technology trends, and respondent demographics. The project demonstrates the complete data analytics lifestyle, including data collection, data preparation, dashboard development, visualisation and business reporting.

# Objectives
- Analyze current technology adoption
- Identify future technology trends
- Explore respondent demographics
- Develop interactive dashboards using IBM Cognos Analytics
- Present findings through data storytelling

# Tools Used
- IBM Cognos Analytics
- Microsoft PowerPoint
- Microsoft Excel
- GitHub

# Data Collection Methods
- Web scraping
- API-based data collection
- Dataset exploration and validation

# Data Preparation 
the following data wrangling methods were performed for this project:
- Duplicate detection
- Duplicate removal
- Missing value identification
- Missing value imputation
- Data normalization
- Data quality validation

# Dashboard sections
Current Technology Usage

Analysis of:
    - Programming Languages Currently Used
    - Databases Currently Used
    - Platforms Currently Used
    - Web Frameworks Currently Used

Future Technology Trends

Analysis of:
    - Programming Languages Desired
    - Databases Desired
    - Platforms Desired
    - Web Frameworks Desired

Demographics

Analysis of:
    - Age Distribution
    - Geographic Distribution
    - Education Levels
    - Age by Education Level

# Key Findings
- Current technology usage is concentrated among a small number of mature technologies.
- Future technology preferences indicate growing interest in emerging and cloud-oriented technologies.
- Respondents represent a diverse global population with varying educational backgrounds and experience levels.
